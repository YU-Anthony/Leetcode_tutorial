# Problem
Given an integer array of even length arr, return true if it is possible to reorder arr such that arr[2 * i + 1] = 2 * arr[2 * i] for every 0 <= i < len(arr) / 2, or false otherwise.

Example 1
```
Input: arr = [3,1,3,6]
Output: false
```

Example 2
```
Input: arr = [2,1,2,6]
Output: false
```
Constraints
- 2 <= arr.length <= 3 * 104
- arr.length is even.
- -105 <= arr[i] <= 105
# Answer
这道题和 2007 很像
```python
class Solution:
    def canReorderDoubled(self, arr: List[int]) -> bool:
        if len(arr) % 2 != 0:
            return False
        res = Counter(arr)
        
        for num in sorted(arr, key=lambda x : abs(x)):
            if not res[num]:
                continue
            if not res[2 * num]:
                return False
            res[num] -= 1
            res[2 * num] -= 1
        return True
```
- TC: O(nlogn)
- SC: O(n)
